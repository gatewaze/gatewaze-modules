# Surveys

Create and distribute surveys, collect responses, and analyze results. This module provides a complete survey management system with an admin dashboard for building surveys and reviewing collected data.

## How It Works

The Surveys module lets you create surveys, distribute them to your audience, and collect responses. The admin interface provides a list view of all surveys and a detail view for each survey where you can review individual responses and analyze results. Database tables store survey definitions, questions, and response data. Content categorization is supported for organizing surveys.

## Configuration

This module has no additional configuration settings.

## Features

- `surveys` -- Core survey functionality
- `surveys.create` -- Build and configure new surveys
- `surveys.responses` -- Collect and analyze survey responses

## Dependencies

None.
