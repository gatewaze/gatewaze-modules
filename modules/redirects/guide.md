# Redirects

Manage URL redirects and short links with click tracking and analytics. This module provides a complete redirect management system with an admin dashboard for creating, reviewing, and analyzing link performance.

## How It Works

The Redirects module lets you create and manage URL redirects with built-in click tracking. The admin interface provides pages for managing all redirects, configuring shortcodes (for use in newsletters), reviewing links that need attention, and viewing detailed analytics for individual redirects. Database tables are created via migration to store redirect mappings and click data.

## Configuration

This module has no additional configuration settings.

## Features

- `redirects` -- Core redirect functionality
- `redirects.manage` -- Create, edit, and delete redirects via the admin dashboard
- `redirects.analytics` -- Click tracking and analytics for redirect links

## Dependencies

None.
