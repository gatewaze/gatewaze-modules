# Event Sponsors

Sponsor profiles, sponsorship tiers, booth assignments, and sponsor team management. This module handles all aspects of sponsor relationships for your events.

## How It Works

Event Sponsors adds a **Sponsors** tab to the event detail view in the admin panel. Organizers can create sponsor profiles, define sponsorship tiers (e.g., Gold, Silver, Bronze), assign booth locations, and manage sponsor team members. Sponsor data is stored in dedicated database tables created during module installation.

## Configuration

This module has no configurable settings.

## Features

| Feature Flag | Description |
|---|---|
| `event-sponsors` | Core sponsor profile and listing functionality |
| `event-sponsors.manage` | Create and edit sponsor profiles, tiers, and booth assignments |
| `event-sponsors.teams` | Manage team members associated with each sponsor |

## Dependencies

| Module | Required |
|---|---|
| `events` | Yes |
